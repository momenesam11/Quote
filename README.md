# ✨ Quote of the Day

A simple and clean **Quote Generator** built using **HTML, CSS, and JavaScript**.

The project displays a random quote with its author when the user clicks the **New Quote** button.

---

## 🚀 Live Demo : https://momenesam11.github.io/Quote/

---

## 🚀 Overview

This project was built as a simple JavaScript practice project to improve DOM manipulation and user interaction skills.

It provides a minimal UI where the user can:
- Read a quote of the day
- Generate a new random quote
- View the quote author

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

## ✨ Features

- Simple and clean interface
- Random quote generation
- Author name display
- Beginner-friendly project structure
- Lightweight and fast
